# Privacy Policy

## Overview

Dissolve is a browser extension that helps users hide distracting webpage elements. This extension does not collect, transmit, or store any personal information from users.

## Data handling

- No personal data is collected.
- No browsing history is stored or transmitted.
- No user accounts are required.
- All interactions happen locally in the browser.

## Permissions

Dissolve uses only the permissions required to function as a content script:
- `activeTab`
- `storage`
- `host_permissions` for `<all_urls>` to allow the extension to operate on web pages where the user activates it.
